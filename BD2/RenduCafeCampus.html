III - Étude de marché Café Campus

1. Objectif de l'étude :

L'objectif de cette étude est de comprendre les besoins des étudiants, des professeurs/chercheurs et du personnel en matière d'utilisation des machines à café sur le campus de Paris-Saclay (le plateau). Nous souhaitons également estimer la taille du marché des boissons chaudes et évaluer la concurrence éventuelle.

2. Marché cible :

- Étudiants de Paris-Saclay sur le campus du plateau

- Professeurs/chercheurs

- Personnel de l'université

3. Données démographiques :

La population de Paris-Saclay est d'environ 85,000 habitants, comprenant un nombre significatif d'étudiants, de professeurs/chercheurs et de personnel de l'université. La tranche d'âge de notre marché cible se situe généralement entre 18 et 50 ans.

étudiants : 65 000

enseignant : 9 000

personnel : 11 000

72 % des Français boivent du café tous les jours.

Sur 1000 adultes interrogés, les résultats montrent que si presque tout le monde boit du café (environ 83 %), il existe de subtiles différences entre générations.

Environ un tiers des adeptes du café (34 %) ont commencé à en boire avant 17 ans, et environ 70 % avant 25 ans.

Tout aussi surprenant, notre consommation ne fait qu'augmenter avec l'âge. 54 % des adultes entre 18 et 29 ans boivent au moins 2 tasses par jour en moyenne et ce pourcentage atteint même 71 % pour les 45-60 ans et 77 % après 60 ans.

Goût irrésistible : environ 69 % des participants boivent du café pour son goût.

Coup de fouet : 62 % d'entre nous ont besoin d'un café pour nous mettre en route (74 % parmi les 18-29 ans).

Concentration : plus d'un tiers des sondés (34 %) déclarent mieux arriver à se concentrer avec un café (après 60 ans, c'est moins évident : 23 % seulement).

Effet relaxant : environ 18 % boivent du café pour gérer leur stress. Là aussi, les seniors sont moins convaincus : 9 %.

Régime équilibré : environ 10 % invoquent des raisons de santé pour justifier leur consommation de café.

4. Tendances du marché :

Les étudiants, les professeurs/chercheurs et le personnel consomment du café principalement entre les pauses, les TD, les amphithéâtres, pour rester éveillés et concentrés pendant les cours. Le café est également consommé pour se réchauffer pendant les périodes froides. Des études ont montré que la consommation de café est souvent associée à un mode de vie étudiant chargé et aux exigences académiques.

5. Concurrence :

5.1 Absence de concurrents directs :

Nous constatons qu'il n'existe actuellement aucun concurrent direct dans le marché de la détection de statut des machines à café sur un campus universitaire. Cette absence de concurrents directs nous donne un avantage initial en tant que pionniers dans ce domaine spécifique.

5.2 Potentiels concurrents indirects :

Cependant, nos concurrents potentiels pourraient provenir du secteur des fabricants de machines à café connectées destinées aux particuliers. Des entreprises renommées telles que Philips, Siemens et Samsung sont des acteurs majeurs dans ce domaine. Bien que leur activité principale soit la conception et la fabrication de machines à café pour un usage domestique, ils pourraient étendre leurs offres pour inclure des fonctionnalités de connectivité et de suivi d'état, les rendant indirectement concurrents dans notre marché cible.

6. Faisabilité technique et financière :

Le développement de Café Campus semble réalisable techniquement et ne nécessite pas de financement majeur. Le modèle commercial pourrait inclure la possibilité de récompenses de fidélité pour encourager les utilisateurs à utiliser l'application.

7. Retours d'expérience :

À collecter lors de futures étapes de recherche et de développement de l'application Café Campus.

En conclusion, cette étude de marché initiale nous permet de mieux comprendre les besoins et les comportements des utilisateurs potentiels de Café Campus sur le campus de Paris-Saclay. Les résultats suggèrent un marché prometteur pour une application fournissant des informations en temps réel sur l'état des machines à café et répondant aux besoins de consommation de café sur le campus.

V - Code Java 


            import java.util.HashMap;
            import java.util.Scanner;
    
            public class CafeMachine {
               private HashMap machines;
    
               public CafeMachine() {
                   machines = new HashMap<>();
                   machines.put("moissan", true);
                   machines.put("lieudevie", true);
                   machines.put("puio", true);
               }
    
               public boolean exists(String nom) {
                   return machines.containsKey(nom);
               }
    
               public boolean status(String nom) {
                   return machines.get(nom);
               }
    
               public void update(String nom, boolean statut) {
                   if (this.exists(nom)) {
                       machines.put(nom, statut);
                       System.out.println("Le statut de la machine " + nom + " a ete mis a jour avec succes !");
                   } else {
                       System.out.println("Mise a jour du statut de la machine " + nom + " a echoue.");
                   }
               }
    
               public void affichageMachine() {
                   for (String nom : machines.keySet()) {
                       boolean statut = machines.get(nom);
                       System.out.println(nom + " - Statut : " + (statut ? "En marche" : "Hors Service"));
                   }
               }
    
               public static void main(String[] args) {
                   System.out.println("Bienvenue sur CafeCampus !");
                   System.out.println("Prenez votre pause cafe sans stress avec Cafe Campus !");
    
                   CafeMachine ensembleMachines = new CafeMachine();
    
                   System.out.println("Voici les machines a cafe et leurs statuts que nous mettons a votre disposition ! :");
    
                   ensembleMachines.affichageMachine();
    
                   Scanner sc = new Scanner(System.in);
                   while (true) {
                       System.out.println("Souhaitez-vous modifier le statut d'une machine ? Entrez oui ou non");
                       String oui_non = sc.nextLine().toLowerCase();
                       if (oui_non.equals("non")) {
                           System.out.println("Ok bye");
                           return;
                       }
    
                       Scanner sc1 = new Scanner(System.in);
                       System.out.println("Quelle machine voulez-vous modifier ?");
                       String machine = sc1.nextLine().toLowerCase();
                       if (!ensembleMachines.exists(machine)) {
                           System.out.println("Cette machine n'existe pas!");
                       }
                       if (ensembleMachines.exists(machine)) {
                           ensembleMachines.update(machine, !ensembleMachines.status(machine));
                           System.out.println(" \n");
                           ensembleMachines.affichageMachine();
                       }
                   }
               }
            }
